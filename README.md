# Stock-Data-Analyzer
A beginner-friendly quantitative finance project that analyzes historical stock data using Python. Pulls market data with APIs, calculates returns and moving averages, and visualizes trends with financial charts using pandas, yfinance, and matplotlib. Great introduction to quant trading and financial data analysis.

- return calculations
- financial data visualization

Built using real-world quant development workflows and clean modular Python architecture.

## Features

- Pull historical stock data using Yahoo Finance
- Calculate daily returns
- Compute moving averages
- Visualize stock trends
- Generate professional charts

## Installation

```bash

cd Stock-Data-Analyzer

pip install -r requirements.txt
```
